    今回の公演、舞台は不思議の国!

    不思議の国で巻き起こる、不思議な不思議な物語……。

    さらに、不思議の国にふさわしい多くのアイドルたちも見逃せない!

    金髪の美少女アリスは、白ウサギを追いかけて、さぁどこへ……? アイドルたちが織りなす気まぐれストーリーにあなたも虜になっちゃうかも!
